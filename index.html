Proyectos

Mikrosys

Sistema integral con frontend y backend en Python para configuración, monitoreo y gestión de routers MikroTik mediante su API. Incluye interfaz gráfica y persistencia en JSON.

Speed Control

Aplicación Python para monitoreo en tiempo real del tráfico NAT en routers. Muestra estadísticas de consumo y genera registros CSV automáticos.

BK-ZTE

Sistema modular en Python para realizar copias de seguridad automáticas de configuraciones en equipos ZTE mediante conexión SSH. Implementa almacenamiento JSON y arquitectura escalable.
